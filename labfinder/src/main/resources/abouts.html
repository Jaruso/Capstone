Included in all search results is a portion of the campus map which marks the location of the building that a room is located in.

Search for lab space based on the software that they have.

Each lab space and classroom on campus has a unique set of software that is installed onto the computers. You can use LabFinder to search for a class room that ahs the software you need.

Do you need specific hardware or amenities? No Problem.

With LabFinder you can search for lab space based on the hardware in the rooms. Printers, Projectors, Scanners, whatever you need. LabFinder will search for everything.
